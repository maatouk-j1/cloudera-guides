The deployment is complete once Cloudera Data Engineering has been enabled. What follows is reference material rather than procedure: the sources the guide draws on, the vocabulary it uses, and a working command reference for the FreeIPA server that underpins DNS and Kerberos throughout.

## This appendix contains the following

- **[References Used in Guide](/installations/cdp-7-3-2/appendix/references)** — Appendix A. The five Cloudera Getting Started and product-overview guides the deployment follows.
- **[Glossary of Terms](/installations/cdp-7-3-2/appendix/glossary-terms)** — Appendix B. Seventeen terms used in this document, each with an upstream reference.
- **[Glossary of Acronyms](/installations/cdp-7-3-2/appendix/glossary-acronyms)** — Appendix C. The twenty-seven abbreviations that appear across the guide, plus three further release-note and supported-version references.
- **[FreeIPA Reference](/installations/cdp-7-3-2/appendix/freeipa)** — adding users, generating keytabs, and the DNS and `ipactl` commands used against the IPA server.

The glossary is deliberately partial. The guide states plainly that it "addresses some terms used in this document, for the purposes of aiding understanding" and is "not a complete list of all multi cloud terminology".
